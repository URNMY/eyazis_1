  • Instruction
  • The logical model treats terms in a query as Boolean variables.
  • If there is a term in the document, the corresponding variable takes the value “true”.
  • Assigning weighting coefficients to terms is not allowed.
  • Queries are formulated as arbitrary Boolean expressions that link terms using standard logical operators: and, or, not.
  • The measure of compliance of the request with the document is the retrieval status value (RSV).
  • In the Boolean model, the fetch status is either 1 if the query expression evaluates to true for a given document, or 0 otherwise.
  • All documents with RSV = 1 are considered relevant to the query.
